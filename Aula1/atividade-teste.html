O que é HTML?

A Linguagem de Marcação de Hipertexto (do inglês, HyperText Markup Language), como diz o nome, é uma Linguagem de Marcação presente no desenvolvimento de páginas na internet.

O HTML é utilizado para a estruturação e formatação do conteúdo de páginas da internet.

Ele contém tags (marcações) que tem funcionalidades específicas

Algumas dessas tags são especiais ou fundamentais na estrutura das páginas, como: "html", "head", "title" e "body"

Por que o HTML não é uma linguagem de programação?

O HTML é intitulado uma linguagem de marcação porque sua função é a estruturalização de páginas na web, utilizando marcações.

Sua diferença natural de uma linguagem de programação é que o html não é um script executável e que ele não atua na criação de algoritmos e execução de operações lógicas.
